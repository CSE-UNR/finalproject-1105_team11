Image Editor

Data
Images - 2D array of int
Saved images - array of ints
Colors(Red, Green, Blue) - int
Name of image - string
File choice - char
Saved names of image - array of strings

Behavior
The user uploads an image, then they may edit (crop, dim, resize) the picture. Images are made up of RGB â€“ the system of colors for digital display in computers. Because RGB is integer numbers that represent how much red, green, and blue are in a color, the user can manipulate the brightness, dimness, or color of their chosen image by altering the int. Finally, the user has the choice to save the edited image as a new image into a file.
